IVEO talking tactile map

The purpose of this project was to assist visually impaired students to more easily navigate around the Michigan State University campus. Four teams worked with the Resource Center for Persons with Disabilities.

Developed a state-of-the-art talking map that allows blind and visually impaired students to easily access a map of the entire main campus. The map allows the user to virtually tour the campus hearing the names of buildings, roads and other landmarks as they are touched. The map provides a mode whereby a student may type the name of a campus landmark with the computer providing directional cues for locating the feature.

The team first created an embossed map of campus using RCPD resources. Using ratios to scale maps appropriately was necessary to designing a map to fit on the IVEO touch pad. Utilizing IVEO technology enabled the team to program locations of roads and landmarks corresponding to the emobossed map.

TOUCH ROADS AND BUILDINGS AND THEY TALK!
A single tap relays the name of the landmark, and a double tap provides other interesting facts.
SOUTH NEIGHBORHOOD MAP LAYOUT

The final prototype is an embossed map of the South Neighborhood of Michigan State University campus. Each building and street is labeled along with hazards like tennis courts and train tracks. Specific types of landmarks (i.e. buildings) utilize a different embossing texture to make the map more distiguishable for visually imparired users. This map may be placed on any IVEO touch pad. A program has been written to allow the IVEO technology to speak names of buildings, roads, etc.
